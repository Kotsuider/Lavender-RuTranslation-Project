## Установка
1. Скачайте
2. Распакуйте в папку с игрой.
3. Профит!

## Редактирование патча
Для распаковки используем GARbro.
Получаем папку patch

Для введения изменений используем [VNTextPatch](https://github.com/arcusmaximus/VNTranslationTools) с командой 
```cmd/powershell
VNTextPatch.exe insertlocal patch Lavanda.xlsx output
```

Для запаковки используем Xp3Pack с командой 
```cmd/powershell
Xp3Pack patch 
```
